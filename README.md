# dotfiles 
## To install:
You need to have Nix installed, https://nixos.org/download.html


```
git clone https://github/Saintdoggie/dotfiles
cd dotfiles
./install --update
```
Hopefully this will work.

## Applications
* Zsh as the shell.
* Home-manager for User Enviroment.
* Hyprland as the Window Manager.
