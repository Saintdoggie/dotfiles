# NixOS Configs

To install, run

```
git clone https://github/Saintdoggie/nixOS-dotfiles
make init
make update
```



## Applications
* Zsh as the shell.
* Home-manager for user applications.
* Hyprland as the Window Manager.
* Rofi as the Application launcher.
