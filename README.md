# NixOS Configs

To install, run

```
git clone https://github/Saintdoggie/nixOS-configs
cd nixOS-configs
./install update
```

## Applications
* Zsh as the shell.
* Home-manager for User Enviroment.
* Hyprland as the Window Manager.

## Config Credits
Rofi, Waybar and Hyprland configs are based on Chris Titus's and Linuxmobile's configs
